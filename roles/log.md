## How to use this log

When a new role-specific CV is completed, add a row above with:
- **Role**: Job title or description
- **Company**: Target company
- **Date**: Month/year the CV was tailored
- **Directory**: Folder name under `roles/`
- **Status**: Draft / Submitted / Interview / Rejected / Offer / Withdrawn
- **Notes**: Brief context (what was emphasized, how the opportunity arose, outcome)
